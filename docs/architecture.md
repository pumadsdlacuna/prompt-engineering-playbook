# Prompt Architecture

## EN
A well-designed prompt usually contains:
1. Role definition
2. Context
3. Task
4. Constraints
5. Output format
6. Evaluation criteria (optional)

## ES
Un prompt bien diseñado normalmente contiene:
1. Definición de rol
2. Contexto
3. Tarea
4. Restricciones
5. Formato de salida
6. Criterios de evaluación (opcional)
